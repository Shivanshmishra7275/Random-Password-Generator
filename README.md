# Password Generator

## Project Description
This is a Password Generator project that creates secure passwords of customizable lengths and complexity. The aim is to enhance security by generating strong passwords for users.

## Features
- Generate passwords of varying lengths.
- Include uppercase letters, lowercase letters, numbers, and special characters.
- Option to generate passphrases for easier memorization.

## Installation Instructions
1. Clone the repository:
   ```
   git clone https://github.com/harsh123-code/password-generator.git
   ```
2. Navigate to the project directory:
   ```
   cd password-generator
   ```
3. Install dependencies (if any):
   ```
   npm install
   ```

## Usage Examples
- To generate a simple password:
   ```
   python generate_password.py 12
   ```
- To generate a complex password with special characters:
   ```
   python generate_password.py 16 --special
   ```
